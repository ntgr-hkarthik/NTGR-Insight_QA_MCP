# Tools

Supporting applications used by the QA workbench.

| Folder | Description |
|--------|-------------|
| **`account-navigator/`** | Express app + browser UI for signup, onboarding, devices, Stripe previews, and QA Mongo helpers — mounted by `dashboard/server.js` at `/navigator/`. |

Install dependencies inside each tool folder when setting up the project (see root **`README.md`**).
