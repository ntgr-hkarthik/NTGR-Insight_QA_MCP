Before any license Mongo change: in Insight → Manage Subscriptions, turn Auto renewal off — Cancel Renewal → OK or Yes in the dialog.

Micro-steps: license-status → license-end → devices-lic-exp (or all). Dry run first; inspect DB after each step. failedStep in JSON if validation fails. Then Insight: License Job → Device Job (NHB device flows: run Device Job again at the end — see below).

Scope (checkboxes only): subscription only → license steps. NHB devices only → device step. Both unchecked or both checked → full sequence (license first, then devices).

NHB — Device Job before device Mongo; then jobs after Mongo

Full sequence (license Mongo, then device Mongo): run Device Job in Insight immediately before the device/licExpDate step — not required before pure license updates.

  1. Run Device Job first (#run-device-job-button). Do not change device documents in Mongo until this has completed.
  2. Run this QA Mongo action (updates licExpDate on matching device rows).
  3. Run License Job (#run-license-job-button).
  4. Run Device Job (#run-device-job-button).
  5. Run Device Job again — second Device Job after License Job so portal and Devices finish syncing.
