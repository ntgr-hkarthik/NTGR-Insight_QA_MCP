# custom-mongodb

Placeholder for a **local copy** of the Node **`mongodb`** package (e.g. symlink or `npm install` into this folder) so scripts such as `scripts/db-subscription-grace-expired.js` can resolve `require('mongodb')` without relying on a global install.

If this directory is empty in your clone, install the driver here or use your own MongoDB client — see **`scripts/README.md`** and **`MCP.md`**.
